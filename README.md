## 📌 About the Project

The **General De Jesus College E-Wallet System** is a digital wallet platform designed for the students, faculty, and staff of General De Jesus College. It aims to provide a secure, convenient, and efficient way to manage campus-related financial transactions — from tuition payments to canteen purchases — all within a single platform.


## 🗃️ Database Design

The database design is handled by **Joshtine Nel Tubon**, covering:

- **Normalization** — Ensuring the database is free of redundancy and anomalies
- **ERD (Entity-Relationship Diagram)** — Visual representation of data entities and relationships
- **DFD (Data Flow Diagram)** — Illustrates how data moves through the system
- **Use Case Diagrams** — Describes system functionality from the user's perspective

Database queries and scripts are maintained by **Jastine Manalastas**.
